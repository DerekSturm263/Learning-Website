Unity

When downloading Unity, you must download version 2020.3.37f1 LTS, since that's the version currently installed on the Sno-Isle computers and Unity has issues with transferring files between differing versions. To download the required version of Unity, visit https://unity.com/releases/editor/archive and switch the version to "2020", then scroll down to "2020.3.37" and install through Unity Hub. However, feel free to download the latest version of Unity if you have any projects at home not for school.


Blender

When downloading Blender, you must download version 3.6.3, since that's the version currently installed on Sno-Isle the computers. Blender doesn't have as much of an issue transferring files between versions as Unity, but still better safe than sorry. To download the required version of Blender, visit https://download.blender.org/release/ and click "Blender 3.6/". Afterwards, click "blender-3.6.3" for the device you are downloading Blender for. As with Unity, feel free to install a later version for personal projects.
