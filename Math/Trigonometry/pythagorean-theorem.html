1. What is the Pythagorean Theorem?

The Pythagorean Theorem is an equation for right triangles that indicates the relationship between the lengths of all 3 sides. Specifically, it states that for all right triangles, if you square the length of both legs and add them together, you get the length of the hypotenuse squared.

Let's put that into equation form...

Given:

  • a and b are the lengths of each leg
  • c is the length of the hypotenuse

Then: a2 + b2 = c2

That means that if we know the length of the hyponetuse squared, we can get the length of the hypotenuse by square rooting our number! To apply this to the equation, we can just square root both sides and it will cancel out with the square on the side with the hypotenuse.

So: c = sqrt(a2 + b2)

3. Why do we need the Pythagorean Theorem?

The Pythagorean Theorem can be used to help us solve the distance between two points, which is a crucial function in video game math.

Any two points in 2D space form a right triangle, which means that if we have the positions of 2 objects, we can get the distance between them by using the Pythagorean Theorem.

5. Solving For Other Sides

What happens if we know the length of the hypotenuse, but only one of the legs? Well, lucky for us, we can still use the Pythagorean Theorem to solve for that missing side. If you need to solve for b, simply plug in the values for a and c and solve for b. If you need to solve for a, simply plug in the values for b and c and solve for a.

An easier way to do this is to subtract a2 or b2 on both sides. Subtract a2 on both sides if you want to solve for b and subtract b2 on both sides if you want to solve for a. The following equations are the result:

a2 = c2 - b2

or

b2 = c2 - a2

After square rooting both sides to get a and b on their own, we get the final two equations for that help us solve for a or b. The equation you choose is based on which sides you know. If you know a and c, choose the equation to solve for b. If you know b and c, choose the equation to solve for a.

a = sqrt(c2 - b2)

or

b = sqrt(c2 - a2)

7. Recap

The Pythagorean Theorem is a helpful equation that helps us solve for the length of any side of any right triangle, as long as we know the lengths of the other 2 sides.

There are 3 helpful equations we can keep in mind for solving for any of these sides:

Given:

  • a and b are the lengths of each leg
  • c is the length of the hypotenuse

c = sqrt(a2 + b2)

a = sqrt(c2 - b2)

b = sqrt(c2 - a2)

These equations are all just the Pythagorean Theorem rewritten in different ways to solve for each of the side lengths.
